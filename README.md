## 🚀 Quraşdırma

### Metod 1 — Birbaşa Import (Tövsiyə olunan)

1. [**Buradan yükləyin**](../../releases/latest) → `azerbaijani_language_pack.mcaddon`
2. Yüklənmiş faylın üzərinə **iki dəfə klikləyin**
3. Minecraft avtomatik olaraq açılacaq və paketi import edəcək
4. Oyunda **Parametrlər → Qlobal Resurslar** bölməsinə keçin
5. Paketi **aktivləşdirin** və oyunu yenidən başladın

### Metod 2 — Əl ilə Quraşdırma

1. `.mcaddon` faylını `.zip` olaraq adlandırın
2. Daxilindəki `resource_pack` qovluğunu aşağıdakı yerə kopyalayın:

```
Android:  /sdcard/games/com.mojang/resource_packs/
iOS:      On My iPhone/Minecraft/games/com.mojang/resource_packs/
Windows:  %AppData%\Roaming\.minecraft\resource_packs\
```

3. Oyunu açın → **Parametrlər → Qlobal Resurslar** → Paketi aktivləşdirin

## 📁 Fayl Strukturu

```
azerbaijani_language_pack.mcaddon
└── resource_pack/
    ├── manifest.json          ← Paket məlumatları
    └── texts/
        ├── en_US.lang         ← Əsas dil faylı (Azərbaycan dili)
        ├── az_AZ.lang         ← Azərbaycan lokal faylı
        ├── languages.json     ← Dəstəklənən dillər siyahısı
        └── language_names.json ← Dil adları
```
